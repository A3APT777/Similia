# Similia (CaseBook) — Project

## Vision

Веб-приложение для врачей-гомеопатов. Помогает вести пациентов, проводить приёмы и работать с гомеопатическим реперторием. Система усиливает мышление врача — не заменяет его.

**Core value:** Врач проводит приём в 2 минуты — записал симптомы, нашёл препарат в реперторие, выписал, завершил.

## Current Milestone: Доработки тура

**Goal:** Довести интерактивный тур до идеала — все элементы подсвечиваются, навигация работает, нет дублирования.

## Validated (реализовано)

- ✓ Авторизация — email/password, сессии через cookies
- ✓ Управление пациентами — создание, редактирование, удаление
- ✓ Редактор консультации — жалобы, модальности, психика, назначение
- ✓ Запись на приём — ScheduleButton
- ✓ Мини-реперторий — поиск, анализ, назначение
- ✓ Полный реперторий — 74 000+ рубрик Кента
- ✓ Анкеты — первичная, острая, предконсультационная
- ✓ Опросы самочувствия (followup)
- ✓ Клинический движок — case_state и decision
- ✓ i18n — русский и английский
- ✓ Настройки — расписание, напоминания, правила приёма
- ✓ ЮKassa — приём платежей (290 ₽/мес, 2900 ₽/год)
- ✓ Реферальная система — +7/+14 дней, лимит 180
- ✓ Отправка назначений — /rx/[token] с правилами приёма
- ✓ Единый онбординг — InteractiveTour (32 шага, 6 блоков)
- ✓ Дизайн — брендовые цвета, hover, тени, focus-visible
- ✓ 220 тестов (vitest)
- ✓ Яндекс.Метрика

## Key Decisions

| Решение | Обоснование |
|---------|------------|
| InteractiveTour вместо driver.js | Единая система, навигация между страницами |
| Реферальный бонус при оплате, не при регистрации | Защита от злоупотреблений |
| 1 уровень рефералов (без пирамиды) | Простота, достаточно для старта |
| Предконсультационный опросник отдельно от intake | Разная цель: intake = анамнез, survey = динамика |
| proxy.ts вместо middleware.ts | Формат Next.js 16 |
