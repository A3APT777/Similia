# Similia (CaseBook) — Project

## Vision

Веб-приложение для врачей-гомеопатов. Помогает вести пациентов, проводить приёмы и работать с гомеопатическим реперторием. Система усиливает мышление врача — не заменяет его.

**Core value:** Врач проводит приём в 2 минуты — записал симптомы, нашёл препарат в реперторие, выписал, завершил.

## Current Milestone: v1.1 — Onboarding

**Goal:** Завершить систему обучения новых пользователей — провести врача через весь рабочий процесс с первого входа до завершения консультации.

**Target features:**
- Этап 2: тур по карточке пациента и редактору консультации
- Этап 3: тур по основному реперторию
- Проверка что всё описанное в туре действительно работает

## Requirements

### Validated (уже реализовано)

- ✓ Авторизация — email/password, сессии через cookies
- ✓ Управление пациентами — создание, редактирование, удаление
- ✓ Редактор консультации — быстрый и глубокий режим, симптомы, назначение
- ✓ Запись на приём — ScheduleButton с выбором даты/времени и свободными слотами
- ✓ Мини-реперторий — поиск рубрик, анализ, топ препаратов, назначение
- ✓ Полный реперторий — поиск, анализ, элиминация, сохранение в консультацию
- ✓ Токены — анкета нового пациента, форма после приёма, загрузка фото
- ✓ Клинический движок — автоматический расчёт case_state и предложение решения
- ✓ i18n — русский и английский язык
- ✓ Настройки расписания врача
- ✓ Онбординг этап 1 — обзор дашборда и форма создания пациента (Driver.js)
